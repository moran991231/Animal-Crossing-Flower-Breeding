# Animal Crossing Flower Breeding
The game `Animal Crossing` has a flower breeding system.  
This is a simulator to understand how to calculate probability.

# Support flower types
[ ] TULIP  
[ ] HYASINTHS  
[ ] ROSE  
[ ] ANEMONE (WINDFLOWER)  
[ ] FANJI  
[ ] CHRYSANTHEMUM  
[ ] COSMOS  
[v] LILY
